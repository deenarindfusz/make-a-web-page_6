How the web works

You and your computer connect to the Internet which uses HTTP to connect to servers.
You have to tell the have to tell your computer what you want your website to look like.
Webpages are text documents written is a language called HTML - Hypertext Markup Langauage.

HTML is made up of

  • Text content - what you see - abcde
  • Markup - what it looks like - abcde
  • References to other documents
    • Links to other pages Udacity
    • Images
    • Videos
      Video

We learned a lot about how Markup uses tags, elements, attributes, and sources to create our webpage.
HTML tags are code that describe how a web page is formatted. They are defined by the characters inside a set of brackets. Most tags come in a set of 2, an opening tag < >and a closing tag < / >.
Elements are compiled of opening tags, content and closing tags.
Attributes provide additional information about an element.
Source elements are used to define the media resource for your pictures and videos.

HTML documents are made up of more than tags. They contain a head and body sections. The head contains metadata and other random stuff.

Before you start to code your website, visualize what you want it to look like.
Draw it out on a piece of paper to get an idea of what it will look like.
This way you can make sure the site will look like you want it.

Boxes

Boxes - everything on your website will be in a box. You have boxes inside of boxes. Things that don't look like a box are a box.
By selecting different attributes of the box you can change how the box looks. It can look like a circle, rectangle or triangle.

CSS Styling in HTML

CSS styling is what is used to select the color, size and styles of the website. It sets the Font color and size, if text is bold or italic.
It sets the shape and color of objects that are loaded into the webite. This is done using attributes and selectors.
Beware of repetition. Repetition is repeating code in places where it has no meaning/value. This makes it difficult to read and troubleshoot what should actually be happening.

3 things you need to remember when coding:

  • Code - Avoid Repetition.
  • Test - Test your code for errors
  • Refine - Correct any errors that found
CSS is a powerful tool. Don't be afraid to play and experiment to find out which styles you want to use on your website.
